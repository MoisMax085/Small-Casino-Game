Game Rules:
  🃏 Blackjack
    Dealer hides 1 card. You get 2, dealer gets 2.
    Winning:
      You get blackjack, dealer doesn’t → You win! 🏆
    Both get blackjack → Draw 🤝
    Dealer gets blackjack, you don’t → You lose 💀
    If no blackjack:
      Keep your cards or draw one more.
      Dealer draws until 17–21.
      Dealer busts → You win! 🎉
